# Sistema de Mercado

Sistema web simples para mercado com:

- login
- cadastro de produtos
- registro de vendas
- relatorio de caixa por dia
- exportacao de vendas em Excel e PDF

## Tecnologias

- HTML, CSS, JavaScript
- Node.js + Express
- SQL Server

## Como executar

```bash
npm install
npm start
```

Abrir no navegador:

`http://localhost:3000`
